## 5. Frontmatter Reference

| Field | Values | Effect |
|---|---|---|
| `hidden: true` | bool | Excludes from all listings; shows "Draft: You've wandered in" banner |
| `draft: true` | bool | Listed publicly; shows "Draft: work in progress" banner; excluded from featured |
| `featured: true` | bool | Shows in ★ Featured slot on homepage (newest wins; draft posts excluded) |
| `featured: { weight: 1 }` | object | Explicit weight for featured ordering (lower wins) |
| `pinned: true` | bool | Floats to top of feed and post listings |
| `authored_by` | `human` \| `ai-assisted` \| `ai-generated` | Authorship badge; omit = Unclassified |
| `updated` | date | "Last updated" date; used for featured sort priority |
| `giscusTerm` | string | Overrides the Giscus comment thread URL (useful for retaining comments after a slug/folder move) |

## 6. Series (Folders)

To group related posts into a cohesive series:
1. Create a subdirectory inside `src/posts/` (e.g. `src/posts/homelab/`).
2. Move your related markdown files into that folder.
3. The Eleventy configuration will automatically create an index page for the folder (e.g. `/blog/homelab/`).
4. The post layout will detect the folder and automatically render a visual "Series Callout" box at the bottom of each post, pointing users to the other articles in the series.
