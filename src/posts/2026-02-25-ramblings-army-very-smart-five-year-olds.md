I started at 3 PM and called it morning.

I've been burning through AI tools faster than my brain can register. Exhausted Claude Code on my Pro subscription, exhausted Codex trial — done. Got access to Copilot CLI - exhausted the 300 premium requests. But the best part was getting access to AWS Bedrock hosted Claude Sonnet 4.6 - no more exhaustion. I finally understand why the creator of Claude code keeps five tabs open at all times.

Five tabs, each split-screened: one side running the agent, one side my own terminal. You just keep switching tabs, approving, continuing the flow. It feeds my ADHD like anything. I'm managing five contexts in my head, staying sharp, and the output doesn't stop.

Here's what actually got done today: debugged my home lab — new CachyOS install, memory segfaults, allocated more memory through the iGPU. Pushed blogs from Obsidian to my website (including this one). Cleaned up my notes. Built a TUI to run and orchestrate all my local agent scripts. Worked on a Chrome extension for an upcoming hackathon. Added features to an ETL pipeline, improved performance on another, ran multiple ad-hoc analyses that went out as emails by end of day.

These agents are the smartest five-year-olds I've ever met. Watching one debug or run analysis — I mean, genuinely something. But give them no baseline and they loop, self-destruct, confidently present you the wreckage. They don't know they need an orchestrator. Right now, the trust factor, the business context — you still need someone who's been in it, knows the code, gives it a foundation to iterate upon and not start from scratch.

"Musicians (agents) play their instruments. I play the orchestra" - an IC-Manager, or a manager-IC?

I don't know where the exponential goes. Maybe it eventually doesn't need the manager at all. Right now it does.

I'm more productive than I've ever been. I'm equally terrified. Both are true at the same time. It's better than sliced bread — I'll go there. It's better than sliced bread, there, I said it.
